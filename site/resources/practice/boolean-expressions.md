# Questions

## Conceptual

What is the result of the following boolean expressions?  

1. `((not False) and True) == True`  
2. `(not False) or (True and False)`  
3. `True and (7 < 3 * 4 / 6)`  
4. `(not False) != True`  
5. `not(True and False) and (False or not False)` 


<details>
  <summary><b>SHOW SOLUTIONS</b></summary>
  

1. `True`  
2. `True`  
3. `False`  
4. `False`  
5. `True`

</details>
